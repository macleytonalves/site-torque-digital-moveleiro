# Site Torque Digital Moveleiro

Landing page premium B2B da Torque Digital para lojas, franquias e marcenarias de móveis planejados.

## Posicionamento central

**Pare de Pagar por Leads. Comece a Pagar por Vendas.**

A Torque estrutura captação, perguntas de proxy, qualificação, CRM, leitura de funil e acompanhamento das decisões comerciais para lojas de móveis planejados.

## Stack técnica

- React
- Vite
- TailwindCSS
- Framer Motion
- SEO técnico
- Sitemap e robots.txt
- Estrutura preparada para GTM, GA4 e Meta Pixel

## Decisão técnica

Projeto criado com arquitetura modular para evitar que ajustes pontuais de fonte, seção, animação ou componente desconfigurem o restante da landing page.
